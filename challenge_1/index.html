Challenge #1: Activating Angular

  1. Copy paste the follwing script tag into the <head> tag:
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.5.8/angular.min.js"></script>
  2. Add the attribute "ng-app" to the html tag, like so:
    <html ng-app>
  3. You should see "Angular is working!" rendered below:
